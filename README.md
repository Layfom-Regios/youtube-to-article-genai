# 🎥 YouTube → Article + PDF Generator (GenAI Project)

Convert any YouTube video into a structured, professional article with downloadable PDF — powered by Gemini AI.

---

## 🚀 Features

- 🔗 Input: YouTube video URL  
- 🧠 AI-powered article generation (Gemini 2.5 Flash)  
- 📄 Clean, structured blog-style content  
- 🌐 HTML webpage generation  
- 📥 Downloadable PDF  
- ⚡ Handles long videos using chunking  

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- Google Gemini API  
- YouTube Transcript API  
- FPDF  

---

## ⚙️ How It Works

1. Extract transcript from YouTube  
2. Clean & preprocess text  
3. Generate article using Gemini  
4. Convert into:
   - HTML webpage  
   - PDF document
